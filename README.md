MArmOS arm analyzer for Space Engineers programmable block.

Idea is to run this script and let it genearate arm definition for you ...

Usage:
1. Paste this script into a programmable block.
2. Run it with the root mechanical block name as the argument.
3. Read the generated MArmOS definition from Custom Data.
4. Replace this script with MArmOS script
5. fill MArmOS_Configuration() with code stored in Custom Data.
6. Profit!

Code is alpha version, and supports only 'simple' arms - no parallel support for now ...
